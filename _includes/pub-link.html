{%- comment -%} Renders one icon-pill link. `include.link.icon` can be one of: - A Phosphor token: "phosphor-lectern", "phosphor-presentation-chart", "phosphor-youtube-logo" → renders from the Phosphor icon font. - An inline-SVG path with the "logo:" prefix: "logo:/Images/trail-logo.svg" → renders . Use this when the wordmark IS the visual signal (lab logos, journal marks). The SVG is expected to be monochrome (single fill) so it blends into the pill chrome. - A Font Awesome class like "fa-file-pdf" → renders . This is the fallback; anything not matched above is assumed to be an FA Solid token. Add new Phosphor tokens to the explicit list. New `logo:` SVG paths just need to live somewhere under /Images/ (or any absolute URL). {%- endcomment -%} {%- if include.link.icon == "phosphor-lectern" -%} {%- elsif include.link.icon == "phosphor-presentation-chart" -%} {%- elsif include.link.icon == "phosphor-youtube-logo" -%} {%- elsif include.link.icon contains "logo:" -%} {%- assign logo_path = include.link.icon | remove_first: "logo:" -%} {%- else -%} {%- endif -%} {{ include.link.label }}
