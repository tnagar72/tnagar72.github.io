{%- comment -%} Renders one icon-pill link. `include.link.icon` can be one of: - A Phosphor token: "phosphor-lectern", "phosphor-presentation-chart", "phosphor-youtube-logo" → renders from the Phosphor icon font. - An inline-SVG path with the "logo:" prefix: "logo:/Images/trail-logo.svg" → renders . Use this when the wordmark IS the visual signal (lab logos, journal marks). The SVG is expected to be monochrome (single fill) so it blends into the pill chrome. - A Font Awesome class like "fa-file-pdf" → renders . This is the fallback; anything not matched above is assumed to be an FA Solid token. Coming-soon pills: when `include.link.url` is empty/missing, the pill renders with `coming-soon` class, no href, and (if a coming_soon_message is set) a data-attribute the site-wide click handler reads to fire a toast. role="button" + tabindex="0" so keyboard/screen-reader users get the same affordance as mouse users. Add new Phosphor tokens to the explicit list. New `logo:` SVG paths just need to live somewhere under /Images/ (or any absolute URL). {%- endcomment -%} {%- assign pending = false -%} {%- if include.link.url == nil or include.link.url == "" -%} {%- assign pending = true -%} {%- endif -%} {%- if include.link.icon == "phosphor-lectern" -%} {%- elsif include.link.icon == "phosphor-presentation-chart" -%} {%- elsif include.link.icon == "phosphor-youtube-logo" -%} {%- elsif include.link.icon contains "logo:" -%} {%- assign logo_path = include.link.icon | remove_first: "logo:" -%} {%- else -%} {%- endif -%} {%- comment -%} Auto-append "(coming soon)" when pending.
